# BIAN v14 Source Directory

This directory is reserved for pinned BIAN v14 raw source artifacts used by
`references/industry_standards/bian/extract-references.py`.

Required files for extraction:

- `BOClassesLite.json`
- `BOEnumerationsLite.json`
- `BusinessAreas.json`
- `BusinessDomains.json`
- `ServiceDomainsBasic.json`

Optional cache directory:

- `cached-classes/` (generated by `--fetch-classes`)

Use `references/industry_standards/bian/source-manifest.md` to record provenance for any
files added here (source URL, retrieval date, checksum).

## Snapshot Acquisition

Use the helper script to download the required files from the version-scoped
BIAN model API endpoints:

```powershell
powershell -ExecutionPolicy Bypass -File .\references\industry_standards\bian\download-snapshots.ps1 -Version 14.0.0
```

If v8 requires authentication, provide a bearer token:

```powershell
$env:BIAN_BEARER_TOKEN = "<token>"
powershell -ExecutionPolicy Bypass -File .\references\industry_standards\bian\download-snapshots.ps1 -Version 14.0.0
```

If upstream returns empty arrays for v14, the script fails by default so
incomplete snapshots are not silently committed.
